## 내일 해야 할 작업

### 1. Ollama 모델 다운로드 (최우선)
모델이 없어서 AI가 실제로 응답하지 못하는 상태.

```powershell
# no-sleep flag 생성 (watchdog이 Ollama 종료 못하게)
New-Item "C:\ProgramData\CHEEZE\backend-agent\no-sleep.flag" -Force

# 모델 pull (6.1GB, 약 10분)
& "C:\Users\zoop7\AppData\Local\Programs\Ollama\ollama.exe" pull huihui_ai/qwen3-vl-abliterated:8b-instruct

# 완료 후 flag 제거
Remove-Item "C:\ProgramData\CHEEZE\backend-agent\no-sleep.flag" -Force
```

> ⚠️ pull 도중 Ollama가 종료되면 처음부터 다시 다운로드됨. no-sleep flag 필수.

### 2. 전체 AI 흐름 테스트
모델 설치 후:
1. Ollama 꺼진 상태에서 메인페이지 AI 질문
2. 진행 표시기 동작 확인 ("Ollama 시작 요청 중..." 단계 전환)
3. AI 응답 도착 확인
4. 5분 후 Ollama 자동 종료 확인

### 3. Gateway 최신 코드 확인
AI queue 코드가 최신 버전으로 올라갔는지 재확인:
```bash
# gateway에서
head -20 /opt/cheeze-ai/cheeze-ai-queue.py | grep TIMEOUT
# OLLAMA_START_TIMEOUT = 120 이어야 함
```

### 4. 백엔드 에이전트 재시작
start_command 전체 경로 변경이 실제 config에 반영됐는지 확인:
```powershell
Stop-ScheduledTask -TaskName "CHEEZE Backend Agent"
Start-Sleep -Seconds 2
Start-ScheduledTask -TaskName "CHEEZE Backend Agent"
```
